## 🚀 Features

- **Live Real-time Data**: Polls Home Assistant via the REST API to display current Watt/kW metrics and daily Energy (kWh) totals.
- **Dynamic Physics Animation**: Power lines glow, pulse, and speed up relative to the amount of power flowing through them.
- **Smart Entity Discovery**: Can't remember your sensor names? Our built-in Smart Discovery tool scans your Home Assistant instance and finds power/energy entities automatically.
- **Drag & Drop Customization**: Use the WordPress admin panel's Live Preview to literally click and drag entity labels and module icons to perfectly fit any background image.
- **Snapshot Backups**: The plugin securely and automatically backs up your configuration settings natively.
- **Extensive Module Support**:
  - 🏠 Home (Load)
  - 🏙️ Grid (Import/Export)
  - ☀️ Solar (PV)
  - 🔋 Home Battery (Charge/Discharge & SOC)
  - 🚗 Electric Vehicle (Charging Power & Battery %)
  - ♨️ Heat Pump (Power, Energy, and COP Efficiency)
  - ☁️ Weather Integration

## 🛠️ Installation

1. Download the latest release `.zip` from the [Releases](https://github.com/chris2172/ha-powerflow/releases) page.
2. In your WordPress Admin dashboard, navigate to **Plugins** -> **Add New** -> **Upload Plugin**.
3. Upload the `.zip` file and click **Install Now**.
4. Click **Activate Plugin**.

## 🔌 Connecting to Home Assistant

To pull live data, the plugin needs read-only access to your Home Assistant server instance. **Your HA instance must be accessible publicly (e.g., via Nabu Casa or a reverse proxy).**

### 1. Generate a Long-Lived Access Token
1. Open your Home Assistant dashboard.
2. Click on your **User Profile Picture** (bottom left corner).
3. Select the **Security** tab.
4. Scroll down to **Long-Lived Access Tokens**.
5. Click **Create Token** and name it something like `WordPress Powerflow`.
6. Instantly copy the generated token string (you won't be able to see it again).

### 2. Configure the Plugin
1. Open your WordPress Admin dashboard and click **⚡ HA Powerflow** in the sidebar.
2. Paste your HA Server URL (including `http://` or `https://` and port if applicable).
3. Paste the **Long-Lived Access Token** into the Access Token field.
4. Click **Test Connection**.

## 🎨 Adding to your Webpage

Once your sensors are linked, inserting the widget into your WordPress site is as simple as adding a shortcode to any post, page, or widget area:

```text
[ha_powerflow]
```

### Drag & Drop Interface
You don't need to be a developer to make the Power Flow widget look exacty how you want. In the HA Powerflow settings, simply use the Live Preview window to drag modules left or right, up or down. Your coordinates update in real-time.

## 🔐 Security Note
We take your server security seriously. Snapshot configuration backups generated by this plugin are heavily encrypted using `openssl_aes-256-cbc` before being saved to your WordPress `uploads` directory. Even if a local file is exposed, your Home Assistant Access Token remains cryptographically secured by your WordPress Salts.
