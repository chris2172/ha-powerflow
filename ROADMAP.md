# HA PowerFlow – Roadmap

This roadmap outlines the planned evolution of the HA PowerFlow WordPress plugin.  
It focuses on improving customisation, visual accuracy, and user‑controlled layout options.

---

## 1. Background Image Management

### ✔ Goal
Allow WordPress admins to upload and manage custom dashboard backgrounds stored in:



### 🔧 Planned Enhancements
- Add support for multiple background presets
- Add a “Remove Background” and “Reset to Default” button
- Add automatic renaming (e.g., `dashboard-bg.png`)
- Add validation (file type, size, dimensions)

---

## 2. User‑Controlled Text Positioning

### ✔ Goal
Because backgrounds may vary, allow users to reposition text elements so they align correctly.

### 🔧 Planned Enhancements
- Drag‑and‑drop positioning for:
  - Grid import/export labels
  - Solar/battery/EV labels
  - Power values
- Save positions to plugin settings
- Add a visual editor mode inside the admin panel
- Add snapping/gridlines for precise alignment

---

## 3. Powerflow Line Repositioning

### ✔ Goal
Allow WordPress admins to reposition the animated SVG powerflow lines so they match the new background layout.

### 🔧 Planned Enhancements
- Drag‑and‑drop control points for SVG paths
- Live preview of animation direction
- Ability to adjust:
  - Start/end points
  - Curve tension
  - Line thickness
  - Animation speed
- Save custom SVG geometry to plugin settings

---

## 4. Future Expansion Ideas

These are not scheduled but are under consideration:

- Multiple dashboard layouts (compact, wide, mobile‑first)
- Theme support (light/dark/transparent)
- Custom icon uploaders (solar, battery, EV, grid)
- Multi‑battery or multi‑inverter support
- Tariff‑aware grid cost visualisation
- Export/import dashboard presets

---
