# HA PowerFlow – Documentation Overview

This folder contains technical and feature documentation for the HA PowerFlow WordPress plugin.

## Contents

- `BACKGROUND-UPLOAD.md`  
  How background uploads work and how the plugin stores custom images.

- `TEXT-POSITIONING.md`  
  Planned system for allowing users to reposition text elements on the dashboard.

- `POWERFLOW-POSITIONING.md`  
  Planned system for repositioning animated SVG powerflow lines.

- `ARCHITECTURE.md`  
  Internal structure of the plugin, including file layout and data flow.

- `CHANGELOG.md`  
  Version history and notable changes.
