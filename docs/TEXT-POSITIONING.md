# Text Positioning System

## Purpose
Custom backgrounds vary, so text elements must be repositionable.

## Planned Features
- Drag‑and‑drop positioning for:
  - Grid import/export labels
  - Solar/battery/EV labels
  - Power values
- Save positions to plugin settings
- Visual editor mode in the admin panel
- Optional snapping/gridlines
- Reset-to-default layout button

## Technical Notes
- Positions stored as percentages for responsive scaling
- Editor mode overlays draggable handles on the dashboard preview
- Uses HTML absolutely‑positioned elements or SVG text nodes
