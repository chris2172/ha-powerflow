== Description ==
A real-time animated power-flow dashboard for Home Assistant using CSS motion paths and SVG. Displays PV, battery, grid, load, and EV flows with direction-aware animation.

== Installation ==
1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through the Plugins menu
3. Go to Settings → HA PowerFlow
4. Enter your Home Assistant URL and Long-Lived Access Token
5. Add the shortcode `[ha_powerflow]` to any page

== Frequently Asked Questions ==
= Does it work with Home Assistant Cloud? =
Yes, as long as you provide a valid URL and token.
